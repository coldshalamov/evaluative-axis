# Research Cycles

Each serious research cycle gets a folder here with six files:

1. `idea.md`
2. `literature.md`
3. `experiment.md`
4. `autopsy.md`
5. `forest.md`
6. `decision.md`

Use the templates in `methodology/templates/`.

The purpose is to force broad thinking at fixed points. A run is not complete
just because it produced a metric. It is complete only after examples are read,
assumptions are challenged, and the Forest Mode memo says what the result means
for the broader research program.

Suggested folder names:

- `cycle_001_intervention_reranking`
- `cycle_002_axis_convergence`
- `cycle_003_cumulative_context_process_scoring`
- `cycle_004_blind_disagreement_adjudication`

Never use these files as performative paperwork. If a template section does not
teach anything, write "not informative" and explain why.
